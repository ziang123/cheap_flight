# cheap_flight

一个基于 Python `tkinter` 的携程低价机票查询小工具，用于批量扫描多个出发城市在指定日期区间内的国内低价航班，并进一步补充对应返程最低价，方便快速比较往返总价。

## 功能概览

- 支持一次输入多个出发城市，批量查询低价航线
- 支持按目的地筛选，留空或输入“全国”即可查询全部国内目的地
- 支持设置去程日期区间和返程日期区间
- 支持设置最高价格上限，优先过滤不符合预算的结果
- 自动为每条去程路线补查返程最低价，并计算往返总价
- 表格展示查询结果，双击即可在浏览器打开携程对应航线页面

## 界面能力

应用提供桌面 GUI，主要输入项包括：

- 出发城市：支持中文城市名或三字 IATA 机场/城市码，多个值用逗号分隔
- 目的地：支持中文城市名或三字码；留空或输入“全国”表示不限制
- 去程日期范围：指定去程搜索窗口
- 返程日期范围：指定返程搜索窗口
- 最高价格：按去程票价做预算过滤
- Cookie：可选，用于提升接口请求成功率

## 运行环境

- Python 3.10 及以上
- Windows 优先，其他支持 `tkinter` 的桌面环境理论上也可运行

## 安装

```bash
pip install -r requirements.txt
```

## 启动

```bash
python app.py
```

## 使用说明

1. 在“出发城市”中输入一个或多个城市，例如 `烟台,威海,青岛`
2. 在“目的地”中输入目标城市；如果希望搜索全国低价票，可留空或输入 `全国`
3. 设置去程和返程日期范围
4. 按需填写最高价格，例如 `800`
5. 如请求不稳定，可补充浏览器中登录后的携程 Cookie
6. 点击查询，等待结果列表刷新
7. 双击结果行可直接打开携程航线详情页

## 输入规则

- 中文城市名与三字码都可以使用
- 多个出发城市请用英文逗号或中文逗号分隔
- 日期格式必须为 `YYYY-MM-DD`
- 最高价格必须为整数

## 实现说明

本项目通过请求携程低价搜索相关接口，获取指定日期区间内的低价航线结果，并在客户端侧继续完成：

- 城市名与 IATA 代码转换
- 去程结果过滤与排序
- 返程最低价并发查询
- 往返总价计算
- GUI 表格展示与跳转

## 注意事项

- 这是一个基于公开网页行为整理的个人工具，接口稳定性取决于目标站点策略
- 若携程接口字段、校验规则或访问策略变更，程序可能需要同步调整
- 高频请求可能触发风控，建议控制查询频率
- `Cookie` 为可选项，请勿将个人敏感凭证提交到仓库或分享给他人

## 常见问题

### 1. 为什么查询失败？

常见原因包括：

- 网络异常
- 携程接口临时变更
- 请求过于频繁触发限制
- 未提供可用 Cookie，导致接口返回不完整或被拦截

### 2. 为什么没有返程结果？

程序会为每条去程路线单独查询返程最低价。如果目标时间段内没有合适返程票，或返程查询被接口限制，对应返程列会显示为空。

### 3. 为什么城市无法识别？

请确认输入的是：

- 标准中文城市名
- 或标准三字 IATA 代码

并避免输入额外空格、括号说明或其他非城市信息。

## 适用场景

- 刷近期低价国内机票
- 对比多个出发地的出行成本
- 估算往返总价，快速筛选高性价比路线
